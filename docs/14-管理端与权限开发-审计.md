## D. 建议优化项

### D-1. 添加 SHA256 密码哈希（IS-14-02）

当前 AdminService.ValidateLogin 做明文比对，建议后续 Sprint 改为：
```csharp
var hash = ComputeSha256(password);
var admin = _db.Admins.FirstOrDefault(a => a.Username == username && a.PasswordHash == hash);
```
同时种子数据改为 `SHA256("123456")`。此项不影响演示功能，属生产化改造。

### D-2. 简化 ValidateLogin 签名（IS-14-04）

当前接口签名 `ValidateLogin(username, password, context)` 在 Service 层直接操作 HttpContext，建议改为只返回 `bool` 或 `(bool, string)`，由 Controller 负责 Session 赋值。

### D-3. 安装 .NET SDK 完成 T02-08 走查（IS-14-03）

建议在本审计结束后安装 .NET 8 SDK，执行：
1. `dotnet build` 验证编译通过
2. `dotnet run` 启动应用
3. T02-08 走查：P06→P07→P08→P09 + 状态回流联动
