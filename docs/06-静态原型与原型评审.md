## 二、视觉方向

本轮采用 **校园服务感** 作为主视觉方向。

- 主色调：深绿 `#2C5F2D`，传递安静、专注、学习的氛围
- 用户端：手机竖屏布局（max-width: 480px），以卡片式信息为主，底部固定操作按钮
- 管理端：电脑端后台布局，左侧固定深色侧边栏 + 右侧白色内容区，以表格和统计卡片为主
- 共享组件：卡片、按钮、状态标签、空状态、提示信息等均通过 `custom.css` 统一定义

## 三、自建样式覆盖说明

除 Bootstrap 5.3 CDN 外，额外编写 `prototype/static-v1/css/custom.css`，覆盖如下内容：
- CSS 变量定义全局色彩、圆角、阴影
- 用户端手机容器 `.mobile-container` 最大宽度 480px
- 管理端布局 `.admin-layout` / `.admin-sidebar` / `.admin-main`
- 自定义按钮 `.btn-primary` / `.btn-outline-primary` / `.btn-lg-block`
- 自定义卡片 `.card-custom` / `.card-custom.clickable`
- 状态标签 `.badge-available` / `.badge-full` / `.badge-disabled` / `.badge-cancelled` / `.badge-active`
- 时段选择 `.time-slot-list` / `.time-slot-item`
- 空状态 `.empty-state`
- 表单控件 `.form-control` / `.form-label`
- 管理端表格 `.table-custom`、筛选栏 `.filter-bar`、统计卡片 `.stats-cards` / `.stat-card`

## 五、页面状态覆盖情况

| 状态类型 | 覆盖页面 | 说明 |
|----------|----------|------|
| 成功/正常态 | 全 9 页 | 各页默认展示正常数据 |
| 可约/已满/停用 | P02、P03、P07、P08 | 通过状态标签和时段禁用展示 |
| 已取消态 | P05、P07 | 灰色标签 + 低透明度卡片 |
| 已预约态 | P05 | 绿色标签 + 取消按钮 |

- **空状态**：在 P05（我的预约页）可通过假数据删除模拟空状态场景；在自定义 CSS 中预置 `.empty-state` 类供后续接入
- **异常提示**：在各页按钮点击时通过 `alert()` 模拟提示反馈

## 六、未接入逻辑

本轮为纯静态原型，以下逻辑尚未接入：
- 没有后端 API 调用（所有数据均为静态 HTML 假数据）
- 没有数据库（未连接 SQL Server LocalDB）
- 没有登录鉴权（管理员登录仅为表单样式，未实现 Session/Cookie）
- 没有真实的预约冲突校验
- 没有时段动态计算
- 统计图表为 CSS 手工柱状图，未接入真实数据
